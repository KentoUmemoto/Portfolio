# Portfolio
このポートフォリオです。  
Topページに遊び心があったほうが良いと考え、簡単なターミナルを設置しました。  
コマンドを入力しにくい環境(スマートフォンなど)のことも考え、クリックでも動作するようにしています。  
基本的にはReact + Viteの構成ですが、更新頻度が高くなるであろう部分はmarkdownで書けるようにしました。  
github pagesでホスティングしていて、mainブランチに変更があればgithub actionsでbuildをするようにしています。

## ソースコード
[Github](https://github.com/KentoUmemoto/kentoumemoto.github.io) にあります。

## 使用技術・ライブラリ等
- React
- Vite
- Tailwindcss
- daisyUI
- github pages
- github actions
