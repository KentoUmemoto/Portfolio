# Reminder (制作中)
シンプルなリマインダーアプリケーションです。  
モバイルをメインに考えていて、PWAでのPUSH通知でリマインドを送れるようにする予定です。  
Next.jsをメインに、ORMとしてPrismaを使用しています。
Github Projectsでタスクを管理して、issueを立てて開発しています。  
今後はサーバー再度の実装とFirebase Cloud Messagingを使用したPUSH通知の実装をしていく予定です。

## ソースコード
[Github](https://github.com/KentoUmemoto/Reminder) にあります。


## 使用技術やライブラリ等
### フレームワークやライブラリ
