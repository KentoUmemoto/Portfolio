# Discordのチャット読み上げbot
Discord上で動作するチャット読み上げbotアプリケーションです。  
友人たちが音声通話をしている中で事情があって音声通話ができない人向けに、チャットを合成音声ソフトで読み上げして擬似的なボイスチャットをできるようにするアプリケーションです。  
似たようなアプリケーションはすでにあるのですが、有料であったり無料のものでも動作が不安定であったりします。  
そのためDockerを使用して簡単にセルフホスト可能なアプリケーションにすることを目標に開発しています。  
現在は単純なチャットの読み上げ機能のみですが、今後は音声の切り替えや単語の読みの辞書登録などの機能を実装したいと考えています。

SNS上の友人たちと共同で開発しています。  
Githubで公開しており、ブランチの運用やissueの管理などの勉強にもなっています。
